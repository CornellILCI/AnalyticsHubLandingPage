    Introducing CIAP

    The Crop Improvement Analytics Platform (CIAP) provides a collaborative and custom JupyterHub environment where researchers can seamlessly access, analyze, and share genotype and phenotype data using R and Python programming environments. CIAP also comes pre-loaded with commonly used libraries such as:

    • sommer for mixed model analysis
    • rTASSEL for GWAS and exploratory genetics analyses
    • rPHG for interacting with PHG data
    • tidyverse- and BioConductor-related libraries for common data manipulation and bioinformatics methods
